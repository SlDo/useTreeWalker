# useTreeWalker

Easy use of the TreeWalker API

- Small size < 0.5 Kb
- SSR supports

## Install

```sh
yarn add @sldo/use-tree-walker
# or
npm i @sldo/use-tree-walker --save
```

## Docs:

Full usage documentation - https://dom.spec.whatwg.org/#interface-treewalker
